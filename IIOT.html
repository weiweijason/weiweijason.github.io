AI-driven SOC/NOC Assistant

Modular Security Automation Framework

Vendor-neutral FastAPI / Flask Bootstrap 5 Embeddable UI

Background

  • Modern SOC/NOC teams drown in alerts and heterogeneous logs.
  • Commercial tools (e.g., SOAR, vendor suites) can be tightly coupled to specific hardware.
  • Goal: Deliver a lightweight, software-only assistant that runs at the application layer.

Objectives

  1. Modular Assistant (Backend): FastAPI module with APIs for log ingestion, incident summarization, playbook generation.
  2. Lightweight Frontend: Bootstrap 5 dashboards + chatbot UI, standalone or embeddable.
  3. Threat Intel Integration: Reputation lookups (VirusTotal, AbuseIPDB) via extendable connectors.
  4. Seamless Embedding: Mountable in Flask, FastAPI, or npm-based apps.

Why This Topic

  • Vendor-neutral AI automation without proprietary appliances.
  • Bridges AI & Cybersecurity through NLP summarization for triage.
  • Practical & Deployable: Works with popular frameworks.
  • Scalable Demonstration: Python package/npm module for reusability.

Architecture Overview

  • Core API (FastAPI): /ingest, /summarize, /playbooks, /intel/*
  • Frontend: Bootstrap 5 UI (chat + dashboards), embeddable widget.
  • Connector Layer: VirusTotal, AbuseIPDB, future TI feeds.
  • Packaging: ai_soc_module (pip) + npm widget.

Embedding Example

# FastAPI integration
from ai_soc_module import SOCAssistant
from fastapi import FastAPI

app = FastAPI()
app.include_router(SOCAssistant.router, prefix="/soc")

<!-- Embeddable widget -->
<div id="soc-assistant" data-endpoint="/soc"></div>
<script src="/static/soc-widget.js"></script>
