# 🎮 國小語文詞語配對遊戲

一個適合國小學生的互動式語文學習遊戲，幫助學生透過例句挖空與詞語配對的方式練習國語詞語，提升詞語理解能力。

## 🎯 遊戲特色

- **互動式學習**：透過挖空例句讓學生主動思考詞語用法
- **即時回饋**：答對答錯立即顯示，加強學習印象
- **進度追蹤**：顯示答題進度和得分統計
- **響應式設計**：支援手機、平板和電腦使用
- **美觀介面**：現代化設計，吸引學生注意力
- **遊戲化學習**：星星獎勵系統和等級提升機制
- **智能吉祥物**：小喵老師提供互動式學習指導
- **語音朗讀**：支援題目語音播放，提升學習體驗
- **多重學習模式**：包含每日任務、錯題複習、學習統計等功能
- **成就徽章**：豐富的成就系統激勵持續學習

## 🕹️ 如何遊玩

### 基本遊戲流程
1. 選擇題目數量（10題/15題/20題）
2. 點選「開始遊戲」按鈕
3. 仔細閱讀挖空的例句
4. 使用🔊按鈕可語音朗讀題目
5. 從四個選項中選擇最適合的詞語
6. 查看答題結果和正確答案
7. 點選「下一題」繼續挑戰
8. 完成所有題目後查看總成績

### 進階功能
- **每日任務**：完成每日學習目標獲得獎勵
- **錯題道場**：複習之前答錯的題目
- **學習統計**：查看個人學習進度和成績分析
- **成就收集**：解鎖各種學習成就徽章
- **星星系統**：答對題目獲得星星，累積升級
- **小喵老師**：點擊吉祥物獲得學習建議和鼓勵

## 🔧 技術實作

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **資料格式**: JSON
- **本地儲存**: LocalStorage API（學習進度、統計資料、成就）
- **語音功能**: Web Speech API（語音朗讀）
- **動畫效果**: CSS3 Transform 和 Transition
- **響應式設計**: CSS Grid 和 Flexbox
- **部署**: GitHub Pages 靜態網頁託管
- **架構模式**: 類別導向程式設計（Class-based）

## 📚 題庫管理

題目儲存在 `data/questions.json` 檔案中，格式如下：

```json
{
  "questions": [
    {
      "id": 1,
      "sentence": "他生病了，整個人＿＿＿＿的，一點力氣也沒有。",
      "correctAnswer": "有氣無力",
      "options": ["欣喜", "有氣無力", "沉重", "餓餓"]
    }
  ]
}
```

### 新增題目步驟：
1. 編輯 `data/questions.json` 檔案
2. 按照格式新增題目物件
3. 設定正確答案和三個干擾選項
4. 儲存檔案即可生效

## 🎮 遊戲系統詳細說明

### 星星獎勵系統
- 答對每道題目獲得 1 顆星星 ⭐
- 累積 10 顆星星可升 1 級
- 星星數據自動保存到本地儲存
- 等級提升時會顯示升級動畫

### 每日任務系統
每日會重置三個任務：
1. **答題挑戰**：完成 10 道題目（獎勵：+50 XP）
2. **準確度大師**：答題正確率達到 80%（獎勵：+3 星星）
3. **學習達人**：完成 3 場遊戲（獎勵：神秘獎勵）

### 錯題道場
- 自動記錄答錯的題目
- 提供專門的錯題複習模式
- 掌握錯題後會從複習清單中移除
- 支援錯題統計和分析

### 學習統計
追蹤以下學習數據：
- 總體正確率
- 總答題數
- 遊戲場次
- 學習時間
- 最近 7 天學習情況圖表

### 成就徽章系統
包含 12 種不同的成就徽章：
- 🌟 初學者、星星收集家、星星傳說
- 📈 等級提升、完美表現、題目大師
- 📅 每日學習者、複習達人、閃電俠
- 🏃 馬拉松選手、精準專家、遊戲大師

### 智能吉祥物（小喵老師）
- 提供學習建議和鼓勵話語
- 根據答題情況給予不同回饋
- 支援點擊互動功能
- 在重要時刻提供引導訊息

## 💾 資料管理說明

### 本地儲存內容
遊戲使用瀏覽器的 LocalStorage 儲存以下資料：
- `wordGame_totalStars`：總星星數
- `dailyMissions`：每日任務進度和連續學習天數
- `wrongQuestions`：錯題記錄
- `learningStats`：學習統計資料
- `unlockedBadges`：已解鎖的成就徽章

### 資料隱私
- 所有學習資料僅儲存在用戶裝置本地
- 不會上傳到任何外部伺服器
- 用戶可隨時清除瀏覽器資料來重置進度
- 適合學校和家庭安全使用

### 資料備份建議
- 重要學習記錄建議定期截圖保存
- 換裝置時需重新開始學習歷程
- 建議配合教師記錄系統使用
