### 🚀 本地開發

1. 克隆專案：
```bash
git clone https://github.com/weiweijason/weiweijason.github.io.git
```

2. 開啟本地伺服器（推薦使用 Live Server 擴展）

3. 訪問 `presentation.html` 查看簡報

### 📝 簡報導航說明
- **方向鍵 ← →**: 上一頁/下一頁
- **方向鍵 ↑ ↓**: 垂直導航（子章節）
- **ESC**: 顯示簡報總覽
- **F**: 全螢幕模式
- **Alt + 點擊**: 縮放功能
